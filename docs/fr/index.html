HUGO

  • Hugo es un moderno framework para creación de sitios web de propósito general.
  • Se trata de un generador de sitios estáticos, escrito completamente en Go.

GitHub

  • GitHub es una forja para alojar proyectos utilizando el sistema de control de versiones Git.
  • Se utiliza principalmente para la creación de código fuente de programas de ordenador.
  • El software que opera GitHub fue escrito en Ruby on Rails.

Docker

  • Docker automatiza el despliegue de aplicaciones dentro de contenedores de software
  • automatización de virtualización de aplicaciones en múltiples sistemas operativos.
